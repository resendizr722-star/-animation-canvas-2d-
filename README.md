# 🎨 Animation Canvas 2D

Proyecto desarrollado con **HTML, CSS, JavaScript y Canvas API**, donde se implementa animación de múltiples círculos utilizando **Programación Orientada a Objetos (POO)**.

---

## 📌 Descripción

Esta aplicación permite generar y animar múltiples círculos dentro de un canvas dinámico.
El usuario puede controlar:

* 🔢 Cantidad de círculos
* 📏 Tamaño del canvas (ancho y alto)
* 🎯 Movimiento y rebote dentro de los límites

Además, se incluye un diseño moderno con **Bootstrap** y efecto visual **Glassmorphism**.

---

## 🚀 Características

* Animación fluida con `requestAnimationFrame`
* Círculos con:

  * Tamaño aleatorio
  * Velocidad aleatoria
  * Colores dinámicos
* Rebote en los bordes sin salirse del canvas
* Interfaz moderna con Bootstrap
* Fondo con imagen + efecto glass
* Canvas configurable en tiempo real

---

## 🛠️ Tecnologías utilizadas

* HTML5
* CSS3
* JavaScript (ES6)
* Canvas API
* Bootstrap 5

* Se mueven de forma continua
* Rebotan al tocar los bordes
* Nunca se salen del área visible
